# {{ name }}

A documentation site built with [Fumadocs](https://fumadocs.dev/) and [Next.js](https://nextjs.org/).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v20 or higher
- [pnpm](https://pnpm.io/) (or npm/yarn)

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the documentation.

### Build

```bash
pnpm build
```

### Deploy

This project is ready to deploy to [Vercel](https://vercel.com). Connect your repository for automatic deployments.

## Project Structure

- `app/` - Next.js app router (layouts and pages)
- `content/docs/` - Documentation content (MDX)
- `lib/` - Configuration and utilities
