# Introduction

Welcome to your new documentation site, built with [Fumadocs](https://fumadocs.dev/).

## Getting Started

Edit the content in the `content/docs` directory. Create new pages by adding `.mdx` or `.md` files. The sidebar will automatically update based on your file structure.

## Features

- **MDX Support** - Write JSX in your markdown
- **Search** - Built-in full-text search
- **Dark Mode** - Automatic theme switching
- **Responsive** - Works on all devices

## Next Steps

Check out the [Getting Started](/docs/getting-started) guide to learn more.
