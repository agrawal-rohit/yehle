# Getting Started

This guide will help you get up and running with your Fumadocs documentation site.

## Development

Start the development server:

```bash
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) to view your documentation.

## Building

Create a production build:

```bash
pnpm build
```

## Deployment

This template is ready to deploy to [Vercel](https://vercel.com). Connect your repository and deploy with zero configuration.

For other platforms, run `pnpm build` and serve the `.next` output directory.
